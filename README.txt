=== Description ===
Shortcode provides a very simple way to add some special content to the WP pages, posts and widgets. Using WP shortcode API developers take the advantages of using shortcodes.

Shortcode Creator Easy is a simply organized WP shortcode builder to create custom shortcodes for displaying the shortcode content anywhere in WP pages, posts and widgets as the requirements. The HTML/JavaScript/CSS content which are used several times in the site can be used as shortcode using this plugin. The user can create custom shortcode from the admin panel without doing any hard code.

The plugin provides a very user-friendly facility to create, edit, delete the custom shortcodes just only using a simple form. Disabling a shortcode when needed can also be possible just by only one click.
